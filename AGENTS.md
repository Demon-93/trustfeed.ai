## Phase Tracking Rule (MANDATORY)

**After completing ANY phase or major feature:**

1. Read `phase_status.md` in the project root
2. Update the phase status:
   - Change `⏳ PENDING` or `🔄 IN PROGRESS` to `✅ COMPLETED`
   - Check off all completed items in that phase
   - Update the "Status Summary" section at the bottom
3. Commit changes with message: `docs: update phase status - [Phase Name] completed`
4. Push to GitHub

**This ensures progress is always visible and trackable.**
